Packages: A bunch of code written by someone else for everyone's use

*npm init -y*: Initating a node js application

*npm i express*: to make our server

# METHODS

GET => when we want to fetch data from server
POST => when we want to send data to the server
PATCH => when we want to update the data in the server
DELETE => delete the data in the server

*npm nodemon server.js*: automatically starts our server on any update

*npm i mongoose*: for server connection with mongodb
